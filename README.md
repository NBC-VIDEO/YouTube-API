## 주요 구현 기능

- [x] 효율적인 Fragment UI 관리
- [x] Retrofit을 활용한 YouTube API 데이터 연동
- [x] 홈화면 (HomeFragment)
- [x] 비디오 검색 (SearchFragment)
- [x] 상세 정보 (VideoDetailFragment or VideoDetailAcitvity)
- [x] 마이 페이지 (MyVideoFragment)
